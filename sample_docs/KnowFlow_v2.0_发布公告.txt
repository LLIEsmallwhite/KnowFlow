一、版本概述

KnowFlow v2.0 是面向企业级场景的 RAG 知识库问答平台，基于 LangGraph + Milvus
构建。本次大版本升级引入了动态 RRF 混合检索、自适应三层分块、记忆压缩等核心能力。

二、核心特性

1. 动态 RRF 混合检索
   - 基于查询意图自动调节向量/关键词权重
   - 向量检索依托 Milvus 2.5 集群，支持十亿级数据秒级检索
   - 关键词检索基于 BM25 算法，精准匹配专业术语

2. 自适应三层分块
   - Structural Chunking：识别 Markdown 标题、代码块边界
   - Semantic Chunking：基于 embedding 相似度切分
   - Fixed Chunking：兜底策略，确保每块不超过 512 tokens
   - 支持 Parent-Child 模式，检索用小块、返回用大块

3. 记忆压缩 (Memory Consolidation)
   - LLM 驱动的对话历史摘要
   - Token 感知的自动触发（超过 32K tokens 时压缩）
   - 压缩率最高 90%，保留关键信息不丢失

三、性能指标

+------------------+-----------+------------+
| 指标              | v1.0      | v2.0       |

| RRF 融合准确率    | 78%       | 91%        |
| 单文档处理时间     | 12s       | 4s         |
| 支持最大 Chunk 数 | 100 万    | 5000 万    |
+------------------+-----------+------------+

四、升级指南

从 v1.x 升级到 v2.0 的步骤：
1. 备份现有 PostgreSQL 数据库
2. 拉取最新镜像: docker compose pull
3. 执行数据库迁移: alembic upgrade head
4. 重启服务: docker compose up -d
5. 验证服务健康状态: curl http://localhost:8000/health

五、已知问题
- 超大 PDF（>500 页）建议拆分为多个文档上传
- 图片 OCR 功能将在 v2.1 版本支持
- Windows 环境下 Milvus 需要 WSL2 后端
